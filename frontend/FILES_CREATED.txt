SVELTE 5 RUNES USAGE
====================

All components use modern Svelte 5 syntax:

✓ $state - Reactive state variables
  Example: let count = $state(0);

✓ $derived - Computed/derived values
  Example: let doubled = $derived(count * 2);

✓ $props - Component properties (destructured)
  Example: const { title, count } = $props();

✓ $effect - Side effects and cleanup
  Example: $effect(() => { console.log(value); });

✓ Snippet blocks - Component template composition
  Example: {@render children?.()}

NO OLD REACTIVE DECLARATIONS:
✗ export let prop;
✗ $: derived = value * 2;
✗ <script context="module">

JWT AUTHENTICATION
===================

Token Management
├─ Stored in localStorage under 'auth_token'
├─ Auto-loaded on app init
├─ Auto-injected as 'Authorization: Bearer <token>' header
├─ Cleared on logout

Data Flow
├─ Login → receive token → store in localStorage
├─ Redirect to /
├─ Root +layout.ts checks token
├─ If valid, load user & libraries
├─ If invalid, redirect back to /auth/login
